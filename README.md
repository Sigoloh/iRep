# iRep
O sistema de gestão de republicas

## Overview
Ao contrário do que os filmes de sessão da tarde e as histórias universitárias contam, morar em república não é apenas festa e desorganização. Morar em república é como administrar uma comuna, onde existem contas a serem pagas, almoços a serem organizados, tarefas a serem divididas e várias outras coisas para se gerenciar.

Desde que comecei a viver a vida republicana, nossa gestão é feita por planilhas, o que dificulta a transparência e o acesso aos dados de forma remota, ao clique de um aplicativo. Por isso, decidi usar meus conhecimentos na área de desenvolvimento para criar uma ferramenta centralizada de gestão de repúblicas, ou como dito hoje em dia, em sistemas de cooliving.

Se gostou da ideia, deixa a estrelinha no repositório e acompanhe de perto o desenvolvimento disso tudo.
